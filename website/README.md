# Raizen Website

This is the official download and landing page for the Raizen distributed AI system.

## Downloads
- Android APK: [downloads/apk/app-debug.apk](downloads/apk/app-debug.apk)
- Windows EXE: [downloads/win-unpacked/electron.exe](downloads/win-unpacked/electron.exe)

## Usage
Open index.html in your browser or deploy the contents to a static web host (e.g., GitHub Pages).

## GitHub Pages
To deploy on GitHub Pages, move all files to the root or set the GitHub Pages source to the main branch `/` folder.
